来自美国的Code Combate

       Code Combate是来自于美国一线的编程教育课程，目前学习支持50余种语言，全球有1200万学习者， 社区累计代码10亿行，注册教师90,000余名，学习者已经遍布全球190个国家

       Code Combate是获得了美国大学理事会认证的学习平台，完成学习后可冲抵大学学分。CodeCombat 推出了AP CSP教学大纲并提供相关教学资源，同时支持中文和英文。

高考出现编程题

       编程题目连续多变出现在各地的高考试卷中，在可以预见的未来，编程题将会成为常规的题目。

      浙江省的新高考中，学生已经可以选考《信息技术》，也就是选考计算机科学和编程。

简单易学

      Code Combate基于Python语言设计的学习工具。Python是一种代表简单主义思想的语言，被大量用户所欢迎的、用途广泛的语言。 阅读一个良好的Python程序就感觉像是在读英语的短文一样，它使你能够专注于解决问题而不是去学习复杂的语法。简单易学，阅读简单。Python语言具有优良的移植性、可扩展性、可嵌入性； 具有种类繁多的扩展库，能满足你各类的应用需求。

寓教于乐

       传统方式的编程教学枯燥而复杂，CodeCombat另辟踩径从最基础的内容开始， 将编程融入到真正的实践中，通过课程中设计的不同迷宫关卡来训练学生的逻辑思考和编程思维来寻求迷宫出路， 求解过程中将会熟悉Python的语法以及编程技巧，且寓教于乐。

完善的课程体系

覆盖从小学到大学全年龄段课程，完美对接美国编程教育课程，学习难度从低到高，针对各年龄段都可以提供学习课程。（不过仍然建议高年龄段初学者从计算机科学一开始学习）
